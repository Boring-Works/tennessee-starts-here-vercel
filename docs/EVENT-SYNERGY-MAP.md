**Document Purpose:** Design event-to-event momentum and cross-sell pipelines that maximize attendance and create a "flywheel effect" where each event feeds the next.

## PART 2: ATTENDANCE PIPELINE ANALYSIS

### Event Spacing Assessment

**Problem Identified:** Two critical gaps exist:

1. **August 2-21** — 19-day gap after Back to School (8/1) to Cherokee Heritage (8/22)
2. **June 27 - July 4** — 7-day gap breaks momentum before peak event

**Solution:** Activate concurrent programs during gaps:

- **Twilight on the Mount** (summer Thursdays) fills June/July evenings
- **Blacksmith Workshops** (select Saturdays) provide weekend anchor
- **Behind the Scenes** (last Saturday monthly) creates exclusivity
- **Hearth Cooking** (quarterly) triggers mid-week engagement

**Result:** No calendar "dead zones" for engaged visitors.

## PART 6: DAILY TOUR INTEGRATION AT EACH EVENT

### Strategy: Tours as the "Connective Tissue"

**Problem:** Events are episodic. Daily tours are the constant. How do we link them?

**Solution:** Every event becomes a **tour expansion point**.

#### Tour Integration by Event Type:

**1. SIGNATURE EVENTS (Road to 250, July 4, Stitching Independence, First Families)**

- Event attendees receive a **free guided tour coupon** at check-in
- Tour available same day (if time permits) or within 7 days
- Script: "Welcome! You're standing where government began. Let me show you the story..."

**2. FESTIVALS (Woolly Days, Early Frontier Days, Harvest Fest, Cherokee Heritage)**

- Event is the "main experience," but tour guides have a **"Did you know?" booth**
- Offer 15-min "lightning tours" during slow hours
- Hand out **"Deep Dive" vouchers** for fuller tours on return visit
- Script: "Want the whole story? Book a full tour, and you'll see artifacts in the vault..."

**3. LECTURES**

- Pre-lecture: Host attendees in main building 30 min early for a **"Context Tour"**
- Walking the grounds + explaining the setting
- Script: "Before Dr. Byrd tells you what frontier settlers believed, let me show you _where_ they believed it..."
- Post-lecture: "If you want to see the artifacts from his talk, tours run daily..."

**4. CAMPS & WORKSHOPS**

- Camps are all-day, but **"tour integration" happens via staff narrative**
- Instructors embed tour content into camp activities
- Script: (Blacksmith) "This technique is the same one William Cobb used in 1790. Let me show you his actual forge ledgers after we finish..."
- After-camp: "Summer camps + daily tours = complete frontier education"

**5. RECURRING PROGRAMS (Homeschool Days, Behind the Scenes, Twilight Tours)**

- **Homeschool Days** = Half tour, half hands-on curriculum
- **Behind the Scenes** = Tour of the vault + artifact handling
- **Twilight Tours** = Standalone evening tour (fills calendar gap)

#### Tour Messaging At Events:

**The "First Visit → Tour Conversion" Script:**

At every event, make this pitch:

- **What you saw today:** The event experience (what you _did_)
- **What you missed:** The story context (what it _means_)
- **The invitation:** "Come back for a daily tour to see the artifacts, read the letters, understand the why."

**Example:**

- Woolly Days (attendee cards wool)
- Tour upsell: "Our master interpreter shows you how the Cobb family's wool became cloth. See the actual spinning wheels."

| Metric                      | Target            | Why                         |
| --------------------------- | ----------------- | --------------------------- |
| **First-time visitors**     | 30% of attendance | New customer acquisition    |

### Red Flags (Adjust Pipeline If Triggered):

- **Calendar gap > 10 days** without event or program → activate Twilight, Workshop, or Behind the Scenes
- **Tour upsell < 10%** → event staff need better tour messaging
- **Repeat visitor rate < 40%** → event didn't create next-event hunger
- **First 250 < 150 by May 1** → intensify Road to 250 messaging at all events
