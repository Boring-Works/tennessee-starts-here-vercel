## "Tennessee Starts Here" — Where History Meets Conversion
### January 2026

---

## EXECUTIVE SUMMARY

After analyzing the hero sections across all three projects—**Production** (Tennessee Starts Here), **Sample A** (Remind Tennessee), and **Sample B** (Tennessee Builders)—I've identified a clear synthesis path that elevates the current production hero from "institutional brochure" to "premium conversion engine" while maintaining the gravitas required for a state historic site.

**The Core Problem:** Production's hero is *beautiful but passive*. It presents information but doesn't create urgency, doesn't respond to the visitor, and doesn't guide them toward a decision.

**The Synthesis:** Merge Production's institutional authority with Sample A's conversion mechanics and Sample B's magnetic delight.

### 1.5 STORYBRAND "GUIDE" ALIGNMENT

The hero must position Rocky Mount as the **Guide**, not the hero. The visitor is the hero of their own story.

| Element | Guide Quality | Current Production | Sample A | Synthesis |
|---------|--------------|-------------------|----------|-----------|
| **Empathy** | Understands the hero's problem | ❌ Too institutional | ✅ "Walk the grounds where the West began" | ✅ Acknowledge the journey |
| **Authority** | Has credibility to help | ✅ "Tennessee State Historic Site" | ✅ "Est. 1790" + wax seal | ✅ Keep both |
| **Plan** | Gives clear steps | ❌ "Plan Your Visit" is vague | ✅ "Claim Your Place" is action | ✅ CTA = enrollment |
| **Calls to Action** | Direct + Transitional | ⚠️ Has both but weak | ✅ Strong primary with urgency | ✅ Progress bar + countdown |
| **Stakes** | Shows what's at risk | ❌ None | ✅ "187/250" scarcity | ✅ Add "95 to One" brand |

**THE "95 TO ONE" BRAND OPPORTUNITY:**

Current messaging lacks stakes. The synthesis should introduce:
> "95 counties. One beginning. **Where are you from?**"

This positions the visitor as part of a collective story—they're not just visiting, they're **claiming their place** in Tennessee's origin story.

### 5.3 Magnetic Text Logic

```typescript
// Simplified magnetic effect
const magneticRange = 120; // pixels
const magneticStrength = 0.35;

letters.forEach((letter, i) => {
  const distance = getDistanceToMouse(letter, mouse);
  if (distance < magneticRange) {
    const pull = (1 - Math.pow(distance / magneticRange, 3)) * magneticStrength;
    letter.style.transform = `translate3d(${dx * pull}px, ${dy * pull}px, 0)`;
  }
});
```
