## TL;DR Verdicts at a Glance

| Verdict         | Conditions                                | Icon | Example                  |
| --------------- | ----------------------------------------- | ---- | ------------------------ |
| **Favorable**   | Clear + 55-75°F + dry + light wind        | 🟢   | 68°F, clear, no rain     |
| **Agreeable**   | Partly cloudy + 50-80°F + pleasant        | 🟡   | 70°F, clouds, pleasant   |
| **Temperate**   | Overcast + 45-80°F + dry                  | 🟠   | 58°F, overcast, dry      |
| **Concerning**  | Mixed issues (heat, cold, rain)           | 🔴   | 82°F, rain possible      |
| **Challenging** | Heavy rain/snow or multiple problems      | 🔴   | Rain all day, 48°F       |
| **Hazardous**   | Safety risk (storms, extreme temps, wind) | 🔴   | Thunderstorms, lightning |

## Temperature Decision Tree

```
Temperature Check:

< 20°F        → Hazardous (extreme cold)
20-35°F       → Check: Clear? → Agreeable. Rainy? → Challenging
35-45°F       → Temperate (cool but manageable)
45-55°F       → Check: Clear? → Favorable. Rainy? → Concerning
55-75°F       → Check: Clear? → FAVORABLE. Clouds? → Agreeable
75-85°F       → Check: Clear? → Agreeable. Humid? → Concerning
85-95°F       → Concerning (heat building)
> 95°F        → Hazardous (extreme heat)
```

---

## Precipitation Decision Tree

```
Precipitation Check:

None           → Favor verdict increases by 1 level
Trace/Drizzle  → Keep verdict same
Light rain     → Drop 1 level (Favorable → Agreeable)
Moderate rain  → Challenging
Heavy rain     → Challenging (always)
Snow           → Challenging or Hazardous
Storms         → Hazardous (always, safety first)
```

## Key Details Priority

### When determining key detail for TL;DR:

1. **Safety first** — Storms, extreme heat/cold, dangerous wind
2. **Comfort/planning** — Temperature + high/low, precipitation, wind
3. **Minor details** — Cloud cover, humidity, UV

### When determining Line 2 for Planning Intelligence:

1. **Safety first** — Lightning window, freeze warning, heat advisory
2. **Time-specific** — "Rain ends 2 PM," "Fog lifts 10 AM"
3. **Planning guidance** — Temperature impact, gear needed

## Approved Vocabulary by Category

### Verdicts (ONLY these 6)

- Favorable, Agreeable, Temperate, Concerning, Challenging, Hazardous

### Temperature (context-dependent)

- "dangerously cold" (<20°F)
- "freezing" (20-32°F)
- "quite cold" (32-45°F)
- "cool but comfortable" (45-60°F)
- "moderate and pleasant" (60-75°F)
- "warm" (75-85°F)
- "hot" (85-95°F)
- "dangerously hot" (>95°F)

### Wind (use these exactly)

- "calm" (<3 mph)
- "a light breeze" (3-8 mph)
- "a gentle breeze" (8-13 mph)
- "a moderate breeze" (13-19 mph)
- "a fresh breeze" (19-25 mph)
- "a strong wind" (25-32 mph)
- "gale conditions" (>32 mph)

### Precipitation

- "no rain expected"
- "light shower possible"
- "drizzle expected"
- "rain likely/probable"
- "heavy rain expected"
- "snow expected"

## FORBIDDEN Words

❌ **Never use these:**

- "good," "bad" (use verdict instead)
- "safe," "dangerous" (use period terms: "hazardous," "concerning")
- "optimal," "ideal" (outside period language)
- "slightly," "kind of," "pretty" (be specific)
- "tonight" without clock time (say "by 11 PM")
- "some" (vague — be specific)
- Modern weather terms like "partly cloudy" (use "partly sunny" or "clouds")

✅ **Use these instead:**

- "good" → "Favorable" / "Agreeable" / "Temperate"
- "bad" → "Concerning" / "Challenging" / "Hazardous"
- "safe" → "favorable conditions" or use specific guidance
- "dangerous" → "hazardous" or specific condition ("lightning risk")
- "slightly" → actual number (32°F, not "slightly cold")

## Time Format Rules

| Format     | Example                | Notes                                  |
| ---------- | ---------------------- | -------------------------------------- |
| Clock time | 2 PM, 11 AM            | Use 12-hour format, exact times        |
| Ranges     | 1-3 PM                 | Use when uncertain exact minute        |

## Examples by Quick Lookup

### "It's raining now, 48°F, will it clear?"

**Check:**

- Precipitation: Light rain (< 0.5") + temp 48°F + clearing soon?
- → "Concerning" or "Challenging"

**TL;DR:** "Unsettled morning - rain, 48°F, clearing this afternoon"

**Planning:**

- Icon: 🟠 (Fair, manageable)
- Line 1: "Fair touring weather - rain now, 48°F"
- Line 2: "Rain ends by 2 PM, dry afternoon"

---

### "It's 85°F, sunny, and HOT"

**Check:**

- Temperature: 85°F + clear + high UV
- → "Agreeable" + caution about heat

**TL;DR:** "Favorable day - 85°F, clear, bring water"

**Planning:**

- Icon: 🟡 (Good, but caution)
- Line 1: "Good conditions for touring - 85°F, sunny"
- Line 2: "Intense sun: seek shade midday, bring water"

---

### "Thunderstorms expected this evening"

**Check:**

- Precipitation: Storms (code 95+)
- → ALWAYS "Hazardous" (safety first)

**TL;DR:** "Hazardous evening - thunderstorms 4-8 PM, lightning"

**Planning:**

- Icon: 🔴 (Unsafe)
- Line 1: "Unsafe conditions - thunderstorms expected"
- Line 2: "Lightning risk 4-8 PM, stay indoors"

---

### "Perfect clear day, 70°F, no concerns"

**Check:**

- Clear + 70°F + dry + light wind
- → "Favorable" (ideal)

**TL;DR:** "Favorable conditions - 70°F, clear, no concerns"

**Planning:**

- Icon: 🟢 (Perfect)
- Line 1: "Perfect conditions for touring - 70°F, clear"
- Line 2: "No rain expected through evening"

## Decision Flow (30-Second Process)

1. **Check weather code** → Storms (95+)? → Use "Hazardous"
2. **Check temperature** → Range determines likelihood
3. **Check precipitation** → None/trace/light/heavy?
4. **Combine** → Pick verdict that matches majority
5. **Add key detail** → Why should visitor care?
6. **Count words** → TL;DR ≤15, Planning lines ~10 each
7. **Check tone** → Read aloud, sounds period-appropriate?
8. **Done!**

## Emergency Decision Rules

### If uncertain between two verdicts:

→ Choose the one that's **more conservative** (safer for visitors)

### If weather changing throughout day:

→ Lead with **current condition**, emphasize **trend** in planning line

### If multiple hazards present:

→ Prioritize **safety first** (storms > heat > cold > wind > rain)

### If you hit the word limit:

→ **Cut details, not guidance**. Guidance (what to do) > Context (why)
