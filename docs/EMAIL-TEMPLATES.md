## Usage Guide

### Customization Fields

Replace all bracketed items with actual information:

- `[GUEST_NAME]` — Visitor's name
- `[EVENT_NAME]` — Name of specific event
- `[SPECIFIC_MOMENT]` — Reference to a particular program moment
- `[GOOGLE_REVIEW_LINK]` — Direct Google Review link for Rocky Mount
- `[UNSUBSCRIBE_LINK]` — Unsubscribe/email preference link

### Best Practices

1. **Send timing:** Within 24-48 hours of event completion
2. **Personalization:** Use guest names when possible—it shows care
3. **Tone:** Keep messages warm and genuine, not transactional
4. **Review links:** Always include both TripAdvisor and Google options
5. **Mobile-friendly:** Test rendering on phone screens before sending
6. **Signature:** Always sign from "The Rocky Mount Team" to feel welcoming

### Review Link Notes

- **TripAdvisor:** The link provided is permanent for Rocky Mount's main listing
- **Google:** Update `[GOOGLE_REVIEW_LINK]` with Rocky Mount's direct Google review link
- Both platforms should open in new tabs to maintain email context

### Follow-up

- If no response after 2 weeks, consider a gentle single follow-up
- Thank guests who leave reviews—it builds loyalty and encourages others
- Monitor reviews for feedback opportunities and operational improvements
