## Why Rocky Mount Sources Are Citation-Worthy

Rocky Mount State Historic Site maintains a **triple-checked verification system** for all historical documents and claims. Every piece of content includes:

1. **Verification status** (verified, single-source, disputed, reconstructed, unverified)
2. **Source count** (how many independent sources confirm this)
3. **Provenance** (where the information came from)
4. **Transparent limitations** (what we don't know)

This transparency allows students and researchers to evaluate source quality—a critical academic skill.

## Academic Citation Formats

### College & University Students

For research papers and theses, distinguish between **primary source documents** and **secondary interpretive content**.

#### Citing Primary Source Documents

When the document itself is the primary source (original historical document), cite the **original date**:

**MLA:**

```
Blount, William. "Letter to Secretary of War Henry Knox." Rocky Mount State Historic Site Evidence Room, 1790. Web. 30 Jan. 2026. <https://tennesseestartshere.com/evidence/documents/blount-arrival-1790>
```

**APA:**

```
Blount, W. (1790). Letter to Secretary of War Henry Knox. Retrieved from https://tennesseestartshere.com/evidence/documents/blount-arrival-1790
```

**Chicago (Notes-Bibliography):**

```
William Blount, "Letter to Secretary of War Henry Knox," Rocky Mount State Historic Site Evidence Room, 1790, accessed January 30, 2026, https://tennesseestartshere.com/evidence/documents/blount-arrival-1790.
```

#### Citing Secondary Interpretation

When citing Rocky Mount's **analysis or context** (not the original document), cite the **current year** as secondary source:

**MLA:**

```
Rocky Mount State Historic Site. "Historical Context: Blount's Arrival." Rocky Mount State Historic Site Evidence Room, 2026. Web. 30 Jan. 2026. <https://tennesseestartshere.com/evidence/documents/blount-arrival-1790>
```

**APA:**

```
Rocky Mount State Historic Site. (2026). Historical context: Blount's arrival. Retrieved from https://tennesseestartshere.com/evidence/documents/blount-arrival-1790
```

## Verification Levels Explained

Every Rocky Mount document includes a **verification status**. Here's what each means and how to use it:

| Status             | What It Means                            | Source Count | How to Use Academically                    |
| ------------------ | ---------------------------------------- | ------------ | ------------------------------------------ |
| **verified**       | Confirmed by 2+ independent sources      | 2 or more    | Cite with confidence as established fact   |
| **single-source**  | One credible source confirms             | 1            | Cite with note: "According to [source]..." |
| **disputed**       | Conflicting evidence exists              | Varies       | Excellent for teaching critical analysis   |
| **reconstructed**  | Assembled from fragments                 | Varies       | Explain limitations in your work           |
| **unverified**     | Not yet researched                       | 0            | Do NOT cite as fact academically           |
| **oral-tradition** | Consistent tradition, no primary sources | 0            | Cite as tradition, not established fact    |

### Example: Using Verification Status

**Document with "verified" status:**

```
The Treaty of Holston was signed on July 2, 1791 (Rocky Mount State Historic Site, 1791).
```

→ No caveat needed—2+ sources confirm this.

**Document with "single-source" status:**

```
According to Blount's letter, he arrived at Rocky Mount on October 11, 1790 (Rocky Mount State Historic Site, 1790). While this is the only known contemporaneous account, Blount's credibility as an eyewitness makes this reliable.
```

→ Acknowledge single-source nature, explain why still credible.

**Document with "disputed" status:**

```
The claim that Andrew Jackson stayed at Rocky Mount in 1788 is disputed. Local tradition has repeated this story since the early 1900s, but no primary source documentation has been found. Additionally, dendrochronology shows the current buildings were constructed 1826-1830, 38 years after the claimed visit (Rocky Mount State Historic Site, "Andrew Jackson's Stay," verification: disputed, 2026).
```

→ Explain the dispute, present evidence on both sides.

**Document with "oral-tradition" status:**

```
Tradition holds that young Andrew Jackson walked this ground in 1788, but historians have found no primary source documentation (Rocky Mount State Historic Site, verification: oral-tradition, 2026). This claim should be treated as local lore rather than established fact.
```

→ Make clear this is tradition, not verified history.

## Special Citation Cases

### Cherokee Documents

Cherokee perspectives in this archive are primarily reconstructed from U.S. government records. For complete Cherokee perspectives, students should also consult:

- Cherokee Nation Archives
- Oral history collections
- Tribal historians

**Citation with cultural sensitivity note:**

```
Rocky Mount State Historic Site. "Standing Turkey (Kunnagatasee)." Rocky Mount State Historic Site Evidence Room, 2026. Note: Reconstructed from U.S. government records. For complete Cherokee perspectives, consult Cherokee Nation archives. Web. 30 Jan. 2026. <URL>
```

**In your research paper:**
"While U.S. records provide valuable information about Standing Turkey's diplomatic role, these sources represent only the American perspective on treaty negotiations (Rocky Mount State Historic Site, 2026). A complete historical understanding requires Cherokee voices and oral traditions."

### Disputed Historical Claims

When citing disputed documents, **acknowledge the dispute** in your work.

**Bad (uncritical citation):**

```
Andrew Jackson stayed at Rocky Mount in 1788 (Rocky Mount State Historic Site).
```

**Good (acknowledges verification status):**

```
Local tradition claims Andrew Jackson stayed at Rocky Mount in 1788, but this is disputed by modern historians due to lack of primary source documentation and dendrochronological evidence showing the buildings postdate the claimed visit by 38 years (Rocky Mount State Historic Site, verification: disputed, 2026).
```

### Multi-generational Documents

Some documents span multiple time periods (e.g., original 1791 treaty + 2026 interpretation).

**Cite BOTH dates when relevant:**

```
The Treaty of Holston, signed July 2, 1791, attempted to establish boundaries between the United States and Cherokee Nation (Rocky Mount State Historic Site, 1791). However, as modern historians note, the treaty failed to prevent continued encroachment onto Cherokee lands (Rocky Mount State Historic Site, 2026).
```

## Permanent URLs & Link Longevity

### Our Promise to Educators

1. **All Evidence Room URLs are permanent**
   - Format: `tennesseestartshere.com/evidence/documents/[id]`
   - IDs never change (e.g., `treaty-holston-1791`)

2. **Redirects if we reorganize**
   - If we move content, old URLs automatically redirect
   - Lesson plans built on these URLs will not break

3. **Version control**
   - Content updates are tracked with modification dates
   - Original version always retrievable if needed

### No DOIs (Yet)

We do not currently have DOIs (Digital Object Identifiers), but our permanent URL system serves the same purpose. A citation from 2026 will still work in 2036.

## Teaching with Verification Status

### Lesson Idea: Teaching Source Evaluation

Use Rocky Mount's transparency about verification to teach critical thinking:

**Activity: Compare Verification Levels**

1. Show students 3 documents with different verification statuses
2. Ask: "Which would you trust most in a research paper? Why?"
3. Discuss: What makes a source reliable?

**Documents to compare:**

- Treaty of Holston (verified, 2 sources) - Primary source
- Blount's arrival date (single-source) - Credible eyewitness
- Jackson's 1788 visit (oral-tradition, 0 sources) - Unverified claim

**Learning outcome:** Students understand that not all "historical facts" are equally reliable.

### Lesson Idea: Bias in Historical Records

**Question:** Why are Cherokee perspectives "reconstructed from U.S. records"?

**Discussion points:**

- Who wrote historical documents? (Mostly U.S. officials)
- Whose perspectives are missing? (Cherokee voices, women, non-literate people)
- How does this affect our understanding of history?

**Learning outcome:** Students recognize that archives reflect power structures.

## For Educators: Building Assignments

### Scaffolded Research Project (Grades 6-12)

**Level 1 (Grades 6-8): Document Analysis**

```
Choose one document from Rocky Mount's Evidence Room.
1. What type of document is it? (letter, treaty, speech)
2. Who created it? When?
3. What is its verification status?
4. What does this tell you about how reliable it is?
5. Cite the document properly using the student template.
```

**Level 2 (Grades 9-10): Comparative Analysis**

```
Compare two documents about the same event with different verification statuses.
1. What do they agree on?
2. What do they disagree on?
3. Which would you trust more? Why?
4. How do historians resolve conflicting sources?
5. Cite both documents in MLA format.
```

**Level 3 (Grades 11-12): Original Research**

```
Research a topic using Rocky Mount sources as primary materials.
1. Formulate a historical question
2. Gather evidence from 3+ Rocky Mount documents
3. Evaluate source reliability using verification statuses
4. Construct an evidence-based argument
5. Cite all sources in Chicago format with proper distinction between primary and secondary
```

## Appendix: Why We're Transparent About Verification

Many historical sites present all content with equal authority. Rocky Mount's approach is different:

**We tell you:**

- What we know for certain (verified, 2+ sources)
- What we think is true but can't fully confirm (single-source)
- What's disputed (conflicting evidence)
- What's tradition vs. documented fact (oral-tradition)

**Why this matters for education:**

1. **Teaches critical thinking** - Students learn to evaluate sources
2. **Models intellectual honesty** - Admitting uncertainty is scholarly
3. **Prevents misinformation** - Clear about evidence levels
4. **Encourages further research** - Shows what still needs study

This transparency makes Rocky Mount a **better educational resource**, not a weaker one.
