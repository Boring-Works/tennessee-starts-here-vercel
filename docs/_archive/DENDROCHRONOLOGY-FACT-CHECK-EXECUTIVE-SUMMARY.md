**Claim:** Jackson stayed in earlier Cobb house (pre-1827), now destroyed

**For:**

**Against:**

**Verdict:** Theoretically possible, currently unverifiable

### Option B: False Tradition (STRONG EVIDENCE)

**Claim:** Jackson never stayed at Rocky Mount; tradition is a secondary source error

**For:**

- Zero primary documentation

- Common for frontier properties to acquire famous names falsely
- All sources appear to repeat same unverified claim

**Against:**

- Jackson was definitely in the region
- Cobb's house definitely existed and hosted guests
- Multiple sources agree on the story
- Specificity ("six weeks") suggests some factual basis

**Verdict:** Plausible, contradicts secondary source consensus

### Option C: Location Confusion (MEDIUM EVIDENCE)

**Claim:** Jackson visited nearby Jonesborough (his actual destination), tradition conflated with Rocky Mount

**For:**

- Jonesborough was Jackson's goal (get law license)
- Location confusion common in oral histories
- Rocky Mount only 10-15 miles away

**Against:**

- All available sources specifically say Rocky Mount
- No Jonesborough sources claim Jackson
- Would require secondary source error across multiple sources

**Verdict:** Deserves investigation, insufficient current evidence
