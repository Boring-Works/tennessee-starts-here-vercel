## UpcomingEventCard Copy Templates

### 3+ Weeks Out (Soft Awareness)

**Template:**

```
[Event Title]

[Short Event Description - 1 sentence]

[Approximate date showing weeks out]

[Link: Learn More] or [Link: View Event Details]
```

**Example:**

```
Woolly Days & Colonial Gardening

Watch raw wool become thread. Dig your hands into soil with heirloom
seeds. Spring fiber arts and heritage gardening.

April 25-26 — In about 3 weeks

[Reserve Your Spot]
```

### 2 Weeks Out (Building Interest)

**Template:**

```
[Event Title]

[Event Description - 2 sentences, emphasize what makes it unique]

[Days remaining] • Forecast: [Seasonal weather type]

Limited spots available. [Link: Book Now]
```

**Example:**

```
Stitching Independence

Two days of stars, stripes, and stitches. Saturday: Tennessee's 230th
recognition and America 250 flag dedication. Sunday: Betsy Ross Sip &
Sew—wine, needles, and flag-making.

14 days away • Forecast: June sunshine, perfect for gathering

Early bird pricing ends soon. [Reserve Your Spot]
```

### 7 Days Out (Urgency + Accuracy)

**Template:**

```
[Event Title]

[Description]

This week • [Accurate forecast: High/Low, condition]

This event is special. [Details on limited capacity/unique aspect]

[Book Your Spot]
```

**Example:**

```
Early Frontier Days & Colonial Trade Fair

Musket smoke drifts past canvas tents. Traders hawk pewter, powder,
and handmade goods. The region's largest Revolutionary-era gathering.

7 days away • Forecast: 71°F, clear skies

This is the biggest market on the calendar. First come, first served.

[Get Your Tickets]
```

### 3 Days Out (FOMO + Live Weather)

**Template:**

```
[Event Title] — THIS WEEKEND

[Brief description of experience]

3 days away • Live forecast now accurate: [Specific weather]

Seats are limited. This weekend is your chance.

[Secure Your Spot]
```

**Example:**

```
Colonial Independence Day — THIS WEEK

America turns 250—and you're standing where Tennessee began. The First
250 ceremony reads each enrolled name aloud.

3 days away • Forecast: 75°F, sunny—perfect conditions

This is the day. This is the place. This is the last chance.

[Reserve Now]
```

### Day-of (Live Conditions + Immediacy)

**Template:**

```
TODAY: [Event Title]

[1-sentence description]

NOW • Live conditions: [Exact temp, condition, visibility]

It's happening right now. [Link: Join Us]
```

**Example:**

```
TODAY: Candlelight Christmas

A thousand candles light the grounds. Carolers sing. Cider warms
cold hands.

NOW • 45°F, clear, perfect starlight

The magic is real. Gates open at 5 PM.

[Book Your Spot]
```

## Copy Rules & Guidelines

### DO

- ✓ Lead with value ("Perfect weather for...") before call-to-action
- ✓ Use frontier-speak sparingly (tie to 1791 context)
- ✓ Emphasize the ground, not the buildings ("Where it happened")
- ✓ Use "this week" / "this weekend" for urgency
- ✓ Connect weather to activity ("72°F is perfect for walking the grounds")
- ✓ Use second person ("your visit", "your spot") for personalization
- ✓ Mention forecast accuracy ("Live conditions now accurate")
- ✓ Use STOP→START replacements from brand guide

### DON'T

- ✗ Never say "Cobb Mansion" (use "Cobb House")
- ✗ Never say "Tennessee's first capital" (it was territorial)
- ✗ Never use "Hallowed Ground" (use "Capital Grounds")
- ✗ Never say "Walk the halls" (use "Stand where they stood")
- ✗ Never claim "America's first western capital" (inaccurate)
- ✗ Never use all-caps for emphasis (only normal case)
- ✗ Never compare to competitor sites ("unlike Sycamore Shoals...")
- ✗ Never use stale data ("next week's forecast" on day 8)
- ✗ Never say "come visit" without context (too generic)

### Testing Your Copy

Before publishing, ask:

1. **Does it answer a real question?** ("Should I visit?")
2. **Is it based on accurate data?** (Real weather, confirmed event)
3. **Would I click this?** (Honest gut check)
4. **Does it follow brand voice?** (Check against COPY.md)
5. **Is it short enough?** (<100 words for cards, <280 for social)
