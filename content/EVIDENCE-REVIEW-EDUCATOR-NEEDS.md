# Evidence Room: Educator Perspective Review

**Date:** January 30, 2026
**Reviewed:** Evidence page at `/evidence` and related features (Library, People, Documents)
**Audience:** Teachers K-12, college instructors, homeschool educators
**Focus Areas:** Teaching utility, classroom integration, student engagement, accessibility

---

## Executive Summary

The Evidence Room is a **primary source archive with strong foundational design**, but currently lacks explicit educational infrastructure. It excels at **authentic document presentation** and **visual storytelling** but needs clear **lesson planning tools**, **grade-level scaffolding**, and **download/print workflows** to maximize classroom value.

**Current Strengths:**

- Exceptional document presentation with citation support
- Compelling narrative structure (The Question → The Appointment → Treaty Signers)
- Verified sources with transparent archival citations
- Mobile-responsive design with accessible navigation

**Critical Gaps for Teachers:**

- No lesson plans or learning objectives
- No downloadable/printable document sets
- No grade-level recommendations
- Limited exploration tools for self-directed learning
- No standards alignment (TN standards, Common Core)

## Teaching Utility Assessment

### How Teachers Would Use This (Today)

#### Existing Use Cases

1. **Documentary Research Source**
   - Teachers bring students to Evidence Room as "primary source verification"
   - Good for: essays, debates, research projects
   - Weakness: No guided worksheet or discussion prompts

2. **Timeline Literacy**
   - Timeline section with verified dates works well for history classes
   - Students can trace cause-and-effect from Washington → Blount → Treaty
   - Weakness: No comparative timeline (what else was happening nationally/globally?)

3. **Document Analysis Practice**
   - Individual documents have context panels and source links
   - Full transcriptions available with no paywall
   - Weakness: No scaffolded analysis questions (for grades 6-8 vs. 10-12)

4. **Bias & Perspective Discussion**
   - "About This Archive" section explicitly addresses whose voices are missing
   - Slavery, Cherokee dispossession, women's labor all noted
   - Weakness: No teaching guide for facilitating this conversation

#### Potential Use Cases (Blocked by Missing Features)

1. **Primary Source Analysis Assignment**
   - Need: Downloadable document sets with discussion questions
   - Status: ❌ Not available

2. **Local History Inquiry Project**
   - Need: Grade-appropriate scaffolding (guiding questions for grades 4-5, 6-8, 9-12)
   - Status: ❌ Not available

3. **Document Literacy Across Content Areas**
   - Could use in English (rhetoric, perspective, bias), Social Studies, AP Seminar
   - Need: Aligned learning objectives for each discipline
   - Status: ❌ Not available

4. **Virtual Field Trip Preparation**
   - Could prepare students before visiting the physical site
   - Need: Pre-visit activity packet
   - Status: ❌ Not available

## Grade-Level Appropriateness Analysis

### Currently Optimal For

**High School (Grades 9-12)**

- Complex narrative structure requires sustained attention
- Treaty signatories and diplomatic maneuvering suits AP US History, AP Government
- "About This Archive" bias section works for AP Seminar, Rhetoric
- Advanced readers can handle Blount's letter without simplification

**College/Advanced**

- Graduate history research
- Thesis/dissertation source material
- Regional studies, Early American history seminars

### Challenging For Younger Grades (Without Modification)

**Middle School (Grades 6-8)**

- ✓ Timeline is accessible and visually clear
- ✓ Signer biographies are short and readable
- ✗ Blount's letter uses period language ("residence," "federal authority")
- ✗ No guided questions to build towards understanding
- ✗ No simplified context (who was Washington? what's a treaty?)

**Elementary (Grades 3-5)**

- ✗ Requires adult scaffolding throughout
- ✗ No kid-friendly introduction to key concepts
- ✗ Dense text, minimal visuals for younger learners
- ✓ Could work with a **"What Happened at Rocky Mount?"** simplified pathway

### Missing: Grade-Level Entry Points

The page needs conditional content:

```
Grade 3-5: "Tennessee Founders Story" (simplified)
├─ Who was Governor Blount?
├─ Why did he live at Rocky Mount?
└─ Who lived here before? (Cherokee intro)

Grade 6-8: "How Government Started" (scaffolded)
├─ The Question (Washington's decision)
├─ The Answer (Blount's appointment)
├─ The Treaty (what it meant)
└─ Discussion: Whose voices are we hearing?

Grade 9-12: "Evidence Analysis" (academic)
├─ Full documents with original language
├─ Bias & interpretation section
├─ Assignment ideas (debate, essay, research)
└─ Standards alignment (CCSS, TN standards)
```

## Citation Tools & Academic Integrity

### Current Implementation: Strong Foundation

✓ **CitationExporter Component**

- Supports MLA, APA, Chicago styles
- One-click copy to clipboard
- Passage-level linking (cites specific sections)
- Clear source attribution in all formats

Example MLA (actual output):

```
"Blount to Knox, October 20, 1790." Rocky Mount State Historic Site,
20 Oct. 1790.
https://tennesseestartshere.com/evidence/documents/blount-arrival-1790#glass-windows.
```

### What Students Need (Beyond Current)

❌ **No MLA 9th Edition Support**

- Cited source format is slightly outdated
- Should be: "Blount, William. Letter to Henry Knox, October 20, 1790.
  Tennessee Starts Here, Tennessee State Parks,
  https://tennesseestartshere.com/evidence/documents/blount-arrival-1790."

❌ **No Footnote/Bibliography Format**

- Some teachers require footnotes (Chicago Manual of Style)
- Current tool generates parenthetical citations but not notes

❌ **No Group Project Tools**

- Students collaborating on research need shared cite-tracking
- No way to "collect citations" from multiple documents

❌ **No Plagiarism Prevention**

- No side-by-side quote comparison
- No paraphrase guidance (how to use without plagiarizing)

### Recommended Enhancements

```
CITATION SUPPORT
├─ Update to Latest Standards
│  ├─ MLA 9th Edition (2021)
│  ├─ APA 7th Edition (2020)
│  └─ Chicago Manual of Style 17th Edition (2017)
│
├─ Format Options
│  ├─ Parenthetical citation (already done)
│  ├─ Footnote format (for papers)
│  ├─ Bibliography entry
│  └─ In-text signal phrase format
│
├─ Export Options
│  ├─ Copy to clipboard (already done)
│  ├─ Email to self
│  ├─ Save to BibTeX / Zotero
│  └─ PDF with full citation
│
└─ Student Integrity Tools
   ├─ "How to Quote Without Plagiarizing" guide
   ├─ Paraphrase template
   ├─ Attribution checklist
   └─ Common citation errors to avoid

## Classroom Integration Scenarios

### Scenario 1: Middle School Social Studies (45 min)

**Teacher Goal:** Students understand what a treaty is and who makes government decisions

**Current Barriers:**

- No middle-school-level introduction to Washington and Blount
- No simplified explanation of federal territory
- Timeline requires significant context to make sense
- No discussion questions to guide exploration

**What's Needed:**

- Pre-visit activity: "Who Decides Where the Capital Goes?" (10 min)
  - Visual showing Washington's question, Blount's answer
  - Vocabulary: federal, territory, governor, negotiation
- Guided Evidence Room exploration (20 min)
  - Directed path: Hero Quote → The Question → The Appointment → Timeline
  - Discussion prompt: "Why did Washington choose Blount?"
- Post-visit activity: "You're the Governor" roleplay (15 min)
  - Student pairs negotiate simple treaty agreement

---

### Scenario 2: High School AP Seminar (90 min)

**Teacher Goal:** Students evaluate claims, assess evidence quality, identify bias

**Current Barriers:**

- No explicit analysis framework for the documents
- "About This Archive" is good but needs classroom discussion scaffold
- Missing counterarguments (what might Cherokee records say differently?)
- No assignment that requires rhetorical analysis

**What's Needed:**

- Pre-visit reading: "What Is Bias in Primary Sources?" (15 min)
  - Teach students to identify author, audience, purpose
- Guided analysis (40 min)
  - Claim: "The Treaty of Holston was a fair agreement"
  - Evidence: Document sets showing (a) US perspective, (b) Cherokee perspectives
  - Discussion: What's missing? What does bias look like?
- Assignment (35 min)
  - Write argumentative claim about treaty fairness
  - Support with 2 documents + counterargument
  - Use proper citations

---

### Scenario 3: English Class: Close Reading & Rhetoric (60 min)

**Teacher Goal:** Analyze Blount's letter for rhetorical choices

**Current Barriers:**

- Letter exists but with no rhetorical annotations
- No guided questions about word choice, tone, purpose
- No comparison to modern official letters

**What's Needed:**

- Pre-visit: Model close reading (Washington's question)
- Main activity: Annotate Blount's "Glass Windows" letter
  - Highlight word choices revealing his perspective
  - Note tone (formal? confident? uncertain?)
  - Identify his persuasive strategy (who's he writing to? what does he want them to believe?)
- Extension: Write Blount's letter from a settler's critical perspective

## Accessibility for Diverse Learners

### Current State

✓ **Good:**

- Semantic HTML structure (headings, lists, navigation)
- Keyboard navigation working
- Text has sufficient contrast (dark background, light text)
- Mobile-responsive (works on tablets/phones for research)
- No auto-playing media (students can control pace)

❌ **Missing Accessibility Supports:**

1. **Reading Support**
   - No readability mode (sans-serif, larger text, dark/light toggle)
   - No text-to-speech integration
   - Dense typography for students with dyslexia

2. **Cognitive Support**
   - Complex narrative requires note-taking
   - No built-in annotation tool
   - No bookmark/save feature for returning to documents

3. **ELL Support**
   - Glossary of historical terms not available
   - No simplified parallel text
   - 18th-century language not flagged or explained

4. **Visual Description**
   - Documents are text-only (good for screen readers)
   - But historical images (if added) would need alt text
   - Signer portraits (if added) would need descriptions

### Recommended Additions

```
ACCESSIBILITY FEATURES

Reading Support
├─ Readability Mode
│  ├─ Toggle sans-serif font
│  ├─ Increase line spacing
│  ├─ Text size adjustment (small/normal/large)
│  └─ High contrast toggle
│
├─ Text-to-Speech
│  ├─ Read document aloud (browser native or third-party)
│  ├─ Highlight as you read
│  └─ Playback speed control
│
└─ Annotation Tools
   ├─ Highlight & save notes
   ├─ Bookmark passages for later
   ├─ Create a personal note collection
   └─ Export notes as study guide

Academic Support
├─ Glossary (hover for definitions)
│  ├─ "federal" = government of multiple states united
│  ├─ "treaty" = formal agreement between nations/governments
│  ├─ "ratify" = officially approve/confirm
│  └─ Links to fuller explanations
│
├─ Historical Context Boxes
│  ├─ "What was the Southwest Territory?"
│  ├─ "Who was George Washington?"
│  ├─ "Why were treaties important?"
│  └─ Expandable, not overwhelming
│
└─ Simplified Pathways
   ├─ "Teen Summary" version of each section
   ├─ Modern English translation of period language
   └─ Key takeaway bullets

## Conclusion

The Evidence Room has **exceptional foundation** for classroom use—authentic documents, clear citations, compelling narrative—but needs **explicit educational scaffolding** to become a standard teaching resource.

**Immediate Actions:**

1. Create 3-version educator guide (grades 6-8, 9-10, 11-12)
2. Add PDF download to all documents
3. Tag all documents with relevant TN standards
4. Add glossary of 20-30 historical terms

**Target:** Position Evidence Room as the **go-to primary source resource for Tennessee classroom teachers**, with all materials freely available and standards-aligned.
